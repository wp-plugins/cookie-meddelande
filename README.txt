This plugin is mainly for Swedish WordPress sites, but text in English follows below

Cookie Meddelande är en mycket enkel plugin till WordPress. Pluginen visar ett meddelandefält på botten av sidan där användaren informeras om att sajten använder cookies. Genom pluginens inställningssida (Inställningar -> Cookie Meddelande) kan du anpassa meddelandetexterna samt länken till mer info. 

Pluginen använder cookiechoices.js, utvecklat av Google för att visa meddelandet.



English:
Cookie Meddelande is a very simple WordPress plugin. It displays a cookie message (in Swedish) at the bottom of the homepage utilizing a Javascript developed by Google. On the settings page of the plugin (Settings -> Cookie Meddelande), you can adapt the message texts and the link to more info.
